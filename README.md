# Requez

## Contribute

### Get the projet

```
$ git clone git@github.com:Quezmedia/requez.git
```

### Instal backend dependencies
```
$ bundle install
```

### Install bower
We are using [bower](http://bower.io/) as package manager for front-end dependencies.

```
$ brew install npm
$ npm install -g bower
```

### Update frontend dependencies
```
$ bower update
```

### Then, run

```
$ foreman start
```
